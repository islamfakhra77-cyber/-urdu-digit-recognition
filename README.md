# Urdu Handwritten Digits Recognition (0-5)

## Project Overview
This project implements a 3-layer neural network to recognize Urdu handwritten digits (0-5) using PyTorch.

## Results
| Metric | Value |
|--------|-------|
| Test Accuracy | >80% |
| Best Validation Accuracy | >85% |
| Training Epochs | 50 |
| Classes | 6 (digits 0-5) |
